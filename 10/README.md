### 동기식과 비동기식

- 동기식
    - 데이터가 입력될 때까지 대기한다
    - 서버에서 이용하기 어렵다
- 비동기식
    - Request 요청시 바로 그 다음으로 넘어간다
    - 데이터가 도착할 때 사용한다

### Select(pre) and IOCP / epoll (post)

#### Select

- READ, WRITE 요청에서 자리가 비었는지 매번 확인한다
- 사용 가능한지 확인하고 사용하는 방식
- 간단하게 구현할 수 있다

### IOCP

- Request 만 요청하여 결과값을 가져가고 결과 값은 queue 형태로 쌓아놓는다
- 더 효율적인 방식이다
